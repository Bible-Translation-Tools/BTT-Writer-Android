NotoSansMultiLanguage-Regular.ttf is a merger of these fonts:
    NotoSans-Regular.ttf
    NotoSansGurmukhiUI-Regular.ttf
    NotoSerifGujarati-Regular.ttf
    NotoSansOriyaUI-Regular.ttf

To merge fonts see forked nototools at: 'https://github.com/unfoldingWord-dev/nototools/tree/added_more_noto_fonts'.
Then look at steps to follow in file 'stepsForMerging.txt'
